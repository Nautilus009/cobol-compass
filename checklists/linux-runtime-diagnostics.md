# Linux Runtime Diagnostics Checklist

Use this checklist when diagnosing COBOL application behavior on Linux.

## Collect

- Hostname
- Current date and time
- Working directory
- Process limits
- Environment variables
- Process information
- Open files
- Filesystem usage
- Mount table
- Shared library dependencies

## Review

- Environment variables
- Shared library path
- File permissions
- Temporary directory space
- Core dump settings
- NFS or network filesystem behavior
- Scheduler-provided environment
- Runtime configuration files
- Executable and shared object versions
