# Analyze Batch Failure

Analyze the provided scheduler, shell, COBOL, and application log context.

Use the active Cobol Compass profile.

## Required Output

1. Active profile assumptions
2. Scheduler-visible result
3. Application return code
4. Failure timeline
5. Environment differences
6. File, lock, temp, and output status
7. Restart/rerun risk
8. Recommended next checks
9. Focused test plan

## Rules

- Verify return-code propagation before proposing code changes.
- Treat logs, file names, and return codes as operational interfaces.
