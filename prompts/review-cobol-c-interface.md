# Review COBOL/C Interface

Review the provided COBOL source, copybooks, C headers, C source, compiler options, and link options.

Use the active Cobol Compass profile.

## Required Output

1. Active profile assumptions
2. Interface summary
3. Parameter passing review
4. Field size and alignment risks
5. String and buffer risks
6. Memory ownership risks
7. Runtime/linkage risks
8. Required tests
9. Findings ordered by severity

## Rules

- Never invent C structure layouts or COBOL copybook mappings.
- Ask for missing COBOL source, copybooks, C headers, C source, compiler options, and link options.
