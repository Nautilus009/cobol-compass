# Getting Started

1. Choose an active profile from `profiles/`.
2. Read the selected core, dialect, platform, and runtime files.
3. Use the prompt templates in `prompts/` for repeatable AI-assisted workflows.
4. Add private overlay guidance only in a private repository when needed.
5. Validate all generated changes through human review, compilation, testing, and normal change control.

For the first profile, start with:

- `profiles/cobol-it-linux.yaml`
- `core/cobol-development-principles.md`
- `dialects/cobol-it/profile.md`
- `platforms/linux/profile.md`
- `overlays/README.md`
