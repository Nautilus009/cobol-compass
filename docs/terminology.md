# Terminology

## Core

Reusable COBOL engineering guidance that is not tied to one compiler, operating system, or runtime.

## Dialect Profile

Rules and risks for a specific COBOL compiler or dialect, such as COBOL-IT, IBM Enterprise COBOL, Micro Focus COBOL, or GnuCOBOL.

## Platform Profile

Operating system or runtime platform behavior, such as Linux filesystem semantics, OpenVMS process behavior, IBM mainframe execution environments, permissions, environment variables, signals, and shell execution.

## Runtime Profile

Middleware or operational runtime guidance, such as Tuxedo services, Control-M batch execution, ActiveMQ messaging, or Oracle DB integration.
