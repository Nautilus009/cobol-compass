# ICC Review Checklist

- Is the change minimal?
- Does it preserve nearby style?
- Are spaces used instead of tabs?
- Are COBOL comments full comment lines with `*` in column 1?
- Does every changed paragraph end with a dot?
- Are existing period patterns preserved on untouched lines?
- Are file status, SQL status, or return-code checks preserved?
- Are DISPLAY and free-text patterns consistent with nearby code?
- Is rollback behavior unchanged or explicitly reviewed?
- Are C integration sizes reviewed when C calls are touched?
- Is there a focused test plan?
