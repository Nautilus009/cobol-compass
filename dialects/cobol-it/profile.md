# COBOL-IT Dialect Profile

This profile contains guidance specific to COBOL-IT.

Use it when the active compiler is COBOL-IT or when a codebase uses COBOL-IT runtime behavior.

## Key Concerns

- Source format: fixed format versus free format
- Compiler options and compatibility flags
- File status behavior
- EXTFH and runtime file handler configuration
- Sort behavior
- Runtime environment variables
- C interoperability
- Differences from IBM Enterprise COBOL, Micro Focus COBOL, and GnuCOBOL
