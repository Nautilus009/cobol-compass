# COBOL-IT Debugging

For debugging COBOL-IT programs:

- Capture the exact compile command and runtime command.
- Verify environment variables used by the runtime.
- Reproduce with production-like file permissions and working directories when safe.
- Collect file status values, return codes, and relevant DISPLAY output.
- Preserve failing input files or records when allowed by policy.
- Check for core dumps, signal termination, or runtime diagnostics on Linux.
