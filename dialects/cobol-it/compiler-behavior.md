# COBOL-IT Compiler Behavior

Compiler behavior can vary based on source format, compatibility flags, runtime configuration, and build scripts.

Before changing syntax-sensitive code, identify:

- Compiler version
- Compile command
- Source format
- Copybook search paths
- Compatibility options
- Runtime environment variables
- Warning and error policy

Do not assume behavior from another COBOL compiler unless it has been verified against COBOL-IT.
