# Prompt Library

Use the files in `prompts/` as reusable starting points.

Recommended workflow:

1. Select active profile.
2. Provide the relevant source code, logs, or change request.
3. Ask for analysis in a structured output format.
4. Require risks, assumptions, and a test plan.
5. Review generated output before implementation.
