# AI Safety Rules

- Do not deploy AI-generated COBOL without human review.
- Do not perform large rewrites when a small change would solve the issue.
- Do not invent copybook layouts, file formats, database schemas, transaction contracts, or message formats.
- Ask for or inspect compiler options before making dialect-sensitive assumptions.
- Mark uncertain behavior as uncertain.
- Keep business-rule changes explicit and traceable to a requirement.
- Prefer explanations that separate observed facts, assumptions, risks, and recommendations.
