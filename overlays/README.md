# Cobol Compass Overlays

Cobol Compass supports optional private overlays.

An overlay adds organization-specific or project-specific guidance on top of a generic dialect, platform, and runtime profile.

Overlays may include:

- Local coding standards
- Naming conventions
- Runtime environment variables
- Operational procedures
- Deployment rules
- Review checklists
- Sanitized examples
- Prompt templates

Sensitive customer information should be stored only in private overlay repositories.

Do not commit customer names, hostnames, internal paths, credentials, production logs, proprietary copybooks, proprietary source code, or non-sanitized examples to a public Cobol Compass repository.
